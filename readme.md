# Docker-Flask 
  
dsdA simple docker images contain Python+Flask for Web or Micro-Services Development.

## Dockerfile

```FROM python:3

WORKDIR /usr/src/app

RUN pip install --upgrade pip
COPY requirements.txt ./
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

```

## Requirements.txt

```Flask>=0.12```
